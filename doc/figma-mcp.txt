1. git 설치

git clone https://github.com/sonnylazuardi/cursor-talk-to-figma-mcp.git

2. powershell 켜기
iwr https://bun.sh/install.ps1 -useb | iex

3. powershell에서 cursor-talk-to-figma-mcp 폴더로 이동
cd cursor-talk-to-figma-mcp
bun install
bunx cursor-talk-to-figma-socket

혹은 bunx cursor-talk-to-figma-mcp

plugin : Cursor Talk To Figma Plugin -> connect -> copy mcp.json

ctrl + shft + P -> flutter : select device
