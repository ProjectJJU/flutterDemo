vscode or cursor : file 인코딩 자체를 utf8로 저장해야지 EUC-KR로 저장시 app에서 한글깨짐!!

Cursor 설정 열기: Ctrl + ,
검색창에 "encoding" 입력
utf8로 변경

'git 한글깨짐'

[i18n]
	commitEncoding = utf-8
	logOutputEncoding = utf-8

[core]


*****

'커서 한글깨짐'

Cursor의 상단 메뉴에서 'Help'를 클릭. 

모든 명령 보기: 'Show All Command'를 선택. 

언어 설정: 'Configure Display Language' 옵션을 찾아 클릭

목록에서 '한국어'를 선택.   

재시작: 언어 변경 후, Cursor를 재시작하여 변경 사항을 적용.
